
# Animal Shelter Database

SQL database design for an animal shelter adoption system.

## Overview

This project models the core processes of an animal shelter, including:
- records of animals in the shelter  
- veterinary examinations and treatment  
- adopters and adoption records  

## Structure

- `physical_model.sql` – table definitions, primary and foreign keys, and constraints  
- `sample_data.sql` – example data for testing  
- `access_rights.sql` – access rights  

## Main entities

- Animals  
- Adopters  
- Veterinarians  
- Veterinary records  
- Adoptions  

## Features

- Relational schema with normalized tables  
- Primary and foreign key relationships  
- Integrity constraints (CHECK, NOT NULL, UNIQUE)  
- Sample data for testing and queries  

## Tools

- SQL  
- relational database design  

## Note

This project was created as part of a database systems course and demonstrates 
basic principles of relational database design and implementation.
